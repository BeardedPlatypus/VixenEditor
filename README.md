## Overview

<p align="center"><img src="https://i.imgur.com/dsMO9Ud.png" alt="nTiled" title="nTiled" width=50% /></p>

VixenEditor is a small voxel application written in C++ and DirectX on top of
UWP. At the current moment it is still in its infancy and needs heavy development.
